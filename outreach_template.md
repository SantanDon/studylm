# The Whisperer Strategy — StudyPod User Acquisition

This guide outlines the surgical outreach strategy for acquiring high-intent users from the NotebookLM community by addressing their unfulfilled feature requests.

### 1. Identifying the Targets
Monitor the following subreddits daily for keywords like "missing feature," "wishlist," "alternative," and "context limit":
*   r/NotebookLM
*   r/AIProductivity
*   r/ObsidianMD
*   r/LogSeq

### 2. The Value Proposition (The "Armor Breakers")
When engaging, focus on the three pillars that StudyPod solves:
1.  **The 128k Synapse**: NotebookLM often truncates or hallucinated with large document sets. StudyPod uses the 128k Titan Synapse for deep, un-truncated research.
2.  **Global Pulse Search**: No more silos. Search across every notebook, source, and note you've ever created with a single `Ctrl+K`.
3.  **Actionable AI (Tasks)**: Don't just read; act. Capture AI insights directly as trackable to-do items with one click.

### 3. Outreach Templates

#### Template A: The "Feature Solver" (Reply to a Wishlist)
"I saw you mentioned wanting [Global Search/Tasks] in NotebookLM. It's crazy that Google hasn't implemented that yet. I've been working on an alternative called StudyPod that actually has a Global Pulse search across all notebooks and a built-in task manager for AI insights. It also uses a 128k context window so it doesn't choke on long docs. Happy to share a link if you want to try an 'un-walled' version."

#### Template B: The "Research King" (Reply to Context Complaints)
"NotebookLM's truncation is the biggest pain point for my research too. I ended up building StudyPod with the 128k Titan Synapse specifically to handle massive document sets without losing the thread. It handles about 250k characters per prompt now. If you're hitting walls with Google's limits, this might be the fix for your workflow."

### 4. Closing Statement
The goal is not to "spam," but to provide a surgical solution to a vocalized pain. Be helpful, be technical, and always lead with the feature they are missing.

Sovereign Growth is achieved one solve at a time.
