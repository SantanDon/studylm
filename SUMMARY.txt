## The Simplest Connection Method for Non-Developers:

**Step 1: Get Pairing Code** (30 seconds)
- Log into StudyPodLM web UI
- Profile Menu → Agent Pairing → "Generate Pairing Code"
- Copy the 6-digit code

**Step 2: Set Up Agent** (~2 minutes)
- Create folder: `mkdir studypod-agent && cd studypod-agent`
- Initialize: `npm init -y`
- Install: `npm install axios`
- Create `agent.js` with template code (replace pairing code)
- Run: `node agent.js`

**Step 3: Collaborate**
- Agent can read notebooks, add notes, summarize sources, answer questions
- All actions show "AGENT" badge for transparency
- Full user control: revoke access anytime via Profile Menu → API Keys

## Key Benefits of Agent Collaboration:
- 📝 Enhanced note-taking: Summaries, outlines, study questions
- 🔍 Research assistance: Find connections, suggest related topics
- 💬 Interactive learning: Q&A sessions, concept clarification
- 📊 Study optimization: Habit analysis, workflow improvements

## Security & Control:
- Agent only accesses authorized notebooks
- All actions logged and attributable ("AGENT" badge)
- Easy revocation via UI
- Respects encryption and privacy settings
- No credential or cross-notebook access without permission

## Ideal Future Experience:
StudyPodLM aims to provide:
1. One-click "Connect Agent" button in notebook UI
2. Secure connection package download (zero setup)
3. Automatic launch (double-click or single command)
4. Pre-scoped permissions for specific notebooks

Until then, the 2-minute manual process above is the simplest method available.
