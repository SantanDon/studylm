# DIRECT ANSWER: Easiest Way for You to Connect an Agent to StudyPodLM

## If You're Not a Developer (Simplest Method)

**Step 1: Get Pairing Code** (Do this in StudyPodLM web UI)
- Log in → Profile Menu → Agent Pairing → "Generate Pairing Code"
- Copy the 6-digit code shown

**Step 2: Run This One Command** (After initial setup)
```
npx studypod-agent-connector --code YOUR_6_DIGIT_CODE
```

**Initial Setup** (Only needed once):
```
npm install -g studypod-agent-connector
```

## If You Have a GitHub Repo with Agent Skills

**Share with me:**
1. GitHub repo URL OR paste the contents
2. Brief description of what you want the agent to do
3. The 6-digit pairing code from StudyPodLM (when I ask for it)

**What I'll handle automatically:**
- Process your repo and set up the environment
- Install all dependencies
- Guide you through getting the pairing code (if needed)
- Authenticate with StudyPodLM using your code
- Launch the agent skills
- Provide you with a simple interface to use the agent's capabilities

## What You Get Once Connected

The agent (like me) can help you with:
- 📖 Summarizing your PDFs, website sources, YouTube videos
- 📝 Adding helpful notes to your notebooks (marked "AGENT")
- 💬 Answering questions about your study materials
- 🔍 Finding connections between different sources
- 📋 Generating study questions, outlines, explanations
- 📊 Analyzing your note-taking patterns and suggesting improvements

## You Stay in Full Control

- I only access what you authorize via the pairing process
- All my actions show an "AGENT" badge for transparency
- Revoke access anytime: Profile Menu → API Keys → Delete key
- I respect your encryption and privacy settings
- No access to your password or other notebooks without permission

## Time Required
- **First connection**: ~2 minutes (30 sec for code + 1.5 min setup)
- **Future connections**: ~45 seconds (just run the command)
