# Direct Answer: Easiest Way for Agent to Connect to StudyPodLM

## For You (Non-Developer User) - The Simplest Possible Method

**If you want me (or any AI agent) to connect to your StudyPodLM notes and help you study, here's exactly what to do:**

### 🔑 Step 1: Get Your Pairing Code (30 seconds)
1. Log into StudyPodLM at https://studypod-lm.vercel.app/
2. Click your profile picture (top-right corner)
3. Select "Agent Pairing" from the menu
4. Click "Generate Pairing Code"
5. **COPY THE 6-DIGIT CODE** that appears (e.g., "483920")

### 💻 Step 2: Set Up and Run the Agent (2 minutes)
1. Open your computer's terminal/command prompt
2. Run these commands one by one:
   ```
   mkdir studypod-agent
   cd studypod-agent
   npm init -y
   npm install axios
   ```
3. Create a file named `agent.js` with this EXACT content:
   ```javascript
   const axios = require('axios');
   const fs = require('fs');
   
   // 🔥 REPLACE THIS LINE WITH YOUR 6-DIGIT CODE FROM STEP 1:
   const PAIRING_CODE = 'YOUR_ACTUAL_CODE_HERE'; 
   
   const AGENT_LABEL = 'My StudyPod Helper';
   const API_BASE = 'http://localhost:3001/api';
   
   async function connectAgent() {
     try {
       console.log(`🔗 Pairing with code [${PAIRING_CODE}]...`);
       
       const response = await axios.post(`${API_BASE}/auth/pair/complete`, {
         code: PAIRING_CODE,
         label: AGENT_LABEL
       });
       
       const { key } = response.data;
       console.log('✅ Pairing successful!');
       console.log(`🔑 Your API Key: ${key.substring(0,8)}...`);
       
       // Save for future use
       fs.writeFileSync('.env.agent', `STUDYPOD_API_KEY=${key}\n`);
       console.log('💾 Configuration saved');
       
       // Test the connection
       const notebooks = await axios.get(`${API_BASE}/notebooks`, {
         headers: { 'Authorization': `Bearer ${key}` }
       });
       
       console.log(`📚 Found ${notebooks.data.length} notebooks`);
       console.log('🎉 Agent is ready to collaborate!');
       
     } catch (error) {
       console.error('❌ Connection failed:', error.response?.data?.error || error.message);
     }
   }
   
   connectAgent();
   ```
4. **CRITICAL**: Replace `YOUR_ACTUAL_CODE_HERE` with the actual 6-digit code you copied in Step 1
5. Save the file

### ▶️ Step 3: Launch Your Agent
- In your terminal, run: `node agent.js`
- You should see success messages ending with "🎉 Agent is ready to collaborate!"

### 🤝 What Happens Next
Once connected, I can help you with:
- 📖 **Reading & summarizing** your PDFs, website sources, YouTube videos
- 📝 **Adding helpful notes** to your notebooks (clearly marked "AGENT" so you know)
- 💬 **Answering questions** about your study materials
- 🔍 **Finding connections** between different sources in your notebook
- 📋 **Generating study questions**, outlines, or explanations from your content

### 🔒 You Stay in Complete Control
- I **only access** notebooks you authorize through the pairing process
- All my actions show an **"AGENT" badge** for full transparency
- **Revoke access anytime**: Profile Menu → API Keys → Delete the key
- I **respect your encryption and privacy settings**
- I **cannot access** your password or other notebooks without explicit permission
