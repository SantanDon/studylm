# EASIEST WAY TO CONNECT AN AGENT TO YOUR STUDYPOWERLM NOTES

## QUICK ANSWER (For Non-Developers)

**Step 1: Get Your Pairing Code** (30 seconds)
- Log into StudyPodLM at https://studypod-lm.vercel.app/
- Click your profile picture (top-right)
- Select "Agent Pairing"
- Click "Generate Pairing Code"
- **COPY THE 6-DIGIT CODE** shown

**Step 2: Run This Command** (2 minutes setup, then 45 seconds each time)
```
mkdir studypod-agent && cd studypod-agent
npm init -y
npm install axios
echo "const axios = require('axios'); const fs = require('fs'); const PAIRING_CODE = 'YOUR_CODE_HERE'; async function connect() { try { console.log(`🔗 Pairing with [${PAIRING_CODE}]...`); const { key } = await axios.post('http://localhost:3001/api/auth/pair/complete', { code: PAIRING_CODE, label: 'My Agent' }); fs.writeFileSync('.env.agent', `STUDYPOD_API_KEY=${key}\n`); console.log('✅ Connected! API Key saved.'); const notebooks = await axios.get('http://localhost:3001/api/notebooks', { headers: { 'Authorization': `Bearer ${key}` } }); console.log(`📚 Found ${notebooks.data.length} notebooks. 🎉 Agent ready!`); } catch (err) { console.error('❌ Error:', err.response?.data?.error || err.message); } } connect();" > agent.js
# REPLACE 'YOUR_CODE_HERE' WITH YOUR ACTUAL 6-DIGIT CODE, THEN RUN:
node agent.js
```

## WHAT HAPPENS AFTER CONNECTION
Once connected, I (or any agent) can help you with:
- 📖 **Reading & summarizing** your PDFs, website sources, YouTube videos
- 📝 **Adding helpful notes** to your notebooks (marked "AGENT" for transparency)
- 💬 **Answering questions** about your study materials
- 🔍 **Finding connections** between different sources in your notebook
- 📋 **Generating study questions**, outlines, or explanations from your content
- 📊 **Analyzing your study habits** and suggesting improvements

## YOU REMAIN IN FULL CONTROL
- I **only access** what you authorize via the pairing process
- All my actions show an **"AGENT" badge** so you know what I did
- **Revoke access anytime**: Profile Menu → API Keys → Delete the key
- I **respect your encryption and privacy settings**
- I **cannot access** your password or other notebooks without explicit permission

## TIME REQUIRED
- **First time**: ~2.5 minutes (30 seconds for code + 2 minutes setup)
- **Future connections**: ~45 seconds (just run `node agent.js` after initial setup)
