# EASIEST WAY TO CONNECT AN AGENT TO STUDYPOWERLM

## FOR NON-DEVELOPERS (STEP-BY-STEP)

### ✅ STEP 1: GET YOUR PAIRING CODE (30 SECONDS)
1. Log into StudyPodLM at https://studypod-lm.vercel.app/
2. Click your profile picture (top-right)
3. Go to "Agent Pairing"
4. Click "Generate Pairing Code"
5. **COPY THE 6-DIGIT CODE** (e.g., 483920)

### ✅ STEP 2: SET UP AGENT (2 MINUTES - FIRST TIME ONLY)
In your terminal/command prompt:
```
mkdir my-agent
cd my-agent
npm init -y
npm install axios
```

Create `agent.js` with:
```javascript
const axios = require('axios');
const fs = require('fs');

// 🔥 REPLACE WITH YOUR 6-DIGIT CODE:
const PAIRING_CODE = 'YOUR_CODE_HERE';

async function connect() {
  try {
    console.log(`🔗 Connecting with code [${PAIRING_CODE}]...`);
    const response = await axios.post('http://localhost:3001/api/auth/pair/complete', {
      code: PAIRING_CODE,
      label: 'My Agent'
    });
    const { key } = response.data;
    console.log('✅ Connected! API Key:', key.substring(0,8) + '...');
    fs.writeFileSync('.env.agent', `STUDYPOD_API_KEY=${key}\n`);
    
    // Test connection
    const notebooks = await axios.get('http://localhost:3001/api/notebooks', {
      headers: { 'Authorization': `Bearer ${key}` }
    });
    console.log(`📚 Found ${notebooks.data.length} notebooks`);
    console.log('🎉 Agent ready to help!');
  } catch (err) {
    console.error('❌ Error:', err.response?.data?.error || err.message);
  }
}

connect();
```

### ✅ STEP 3: RUN & USE (45 SECONDS - FUTURE TIMES)
```
node agent.js
```

## WHAT THE AGENT CAN DO FOR YOU:
- 📖 Read & summarize your PDFs, websites, YouTube videos
- 📝 Add helpful notes to your notebooks (marked "AGENT")
- 💬 Answer questions about your study materials
- 🔍 Find connections between different sources
- 📋 Generate study questions, outlines, explanations
- 📊 Analyze your study habits and suggest improvements

## YOU STAY IN CONTROL:
- Agent only accesses what you authorize
- All actions show "AGENT" badge
- Revoke anytime: Profile Menu → API Keys → Delete
- Respects your encryption & privacy
- No access to password or other notebooks
